CSSGrid

CSSGrid is an easier-to-use grid. No .first, no .last, just nestable grid slots.

Purpose

CSSGrid is an effort to use less classes to achieve a comprehensive and flexible grid. Each element in the grid only requires one class. That means no .first or .last classes anywhere.

Nesting

Nesting elements 1 level down, is no problem. No special classes or elements required.

Weight

CSSGrid means less junk in your HTML in favor of a little bit of extra CSS.

How to Use It

Using CSSGrid is easy. It starts with a wrapper which is called .grid in this example (but you could change it to whatever you'd like). Once you've got the wrapper in place, create divs with a class of .row. Inside each row, create a set of columns for content areas.

Columns

CSSGrid supports both 4 and 6 column structures. Each column class is prefixed with a c4 or c6 designator to indicate which column type you are using. Then it's a simple matter of choosing the slot position (or multiple slot positions) you wish to use (1-n).

Spanning columns is done by simply running numbers together, for example: c4-12 would be a 1/2 wide column while c6-56 would be a 1/3 wide column at the end of the 6 column row.
